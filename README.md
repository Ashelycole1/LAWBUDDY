# LAWBUDDY

An intelligent AI based strictly on the Ugandan constitution to help citizens know their rights, law students easily access and pre for exams, pro view for easy reference of different articles via the pro view.

## Features
- AI-powered legal assistance based on the Ugandan Constitution.
- Glassmorphic UI design.
- Pro view for easy reference.
